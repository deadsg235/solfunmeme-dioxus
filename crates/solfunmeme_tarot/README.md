# `solfunmeme_tarot`

This crate provides the core data structures and logic for the Digital Secretome Tarot, a central concept in the `solfunmeme-dioxus` project.

## Features

*   Defines the `TarotCard` enum, representing a standard 78-card Tarot deck.
*   Provides a foundation for integrating Qabalistic correspondences and other symbolic systems.
*   Designed to work with the "Market Maker" metaphor, where cards represent computational resources or market participants.

## Usage

This crate is intended to be used as a foundational component for higher-level analysis and visualization tools within the `solfunmeme-dioxus` ecosystem.
