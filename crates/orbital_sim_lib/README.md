# `orbital_sim_lib`

This crate provides functionalities for simulating orbital mechanics and celestial body interactions.

## Purpose

It enables the Solfunmeme project to model and visualize complex orbital paths, potentially for representing abstract relationships or data flows in a dynamic, spatial context.

## Core Functionalities

-   **Simulate Orbits**: Calculate and predict the trajectories of celestial bodies or abstract entities under gravitational influence.
-   **Collision Detection**: Detect potential collisions between simulated objects.
-   **State Evolution**: Advance the state of the simulation over time.
