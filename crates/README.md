# Crates Overview

This directory contains all the Rust crates that constitute the `solfunmeme-dioxus` project. Each crate is designed with a specific purpose, contributing to the overall functionality of the Code-Math Manifold.

For detailed information on each crate, please refer to its individual `README.md` file located within its respective directory.

## Crate Categories:

### Core Application Crates
These crates form the backbone of the application, managing the UI, workflows, and overall project state.

### Data & Analysis Core
These crates define fundamental data structures and provide core business logic for processing and analyzing data.

### Code Analysis & Processing
Focused on parsing, extracting, and analyzing source code files into structured data.

### Mathematical & Embedding Systems
These crates handle the mathematical transformations, including Clifford algebra operations and BERT embedding generation, central to the Code-Math Manifold.

### Semantic & RDF Processing
Responsible for converting structured data into RDF triples and handling JSON-LD for semantic representation.

### UI & Component System
Contains reusable UI components, state management for the user interface, and tools for building interactive elements.

### Blockchain & Solana Integration
These crates facilitate interaction with the Solana blockchain, including wallet integration, account management, and on-chain data storage.

### NLP & Language Processing Plugins
Provides various natural language processing capabilities, from tokenization and embedding to grammar checking and keyword extraction.

### Search & Indexing
Manages full-text search functionalities and indexing of code and data.

### Storage & I/O
Handles interactions with external storage systems and file operations.

### Scripting & Automation
Integrates scripting engines for automation and custom logic.

### Emoji & Workflow System
Enables the use of emojis for defining and managing workflows.

### Micro-Component Protocol
Contains foundational types and traits for the Micro-Component Protocol.
